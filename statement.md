## Project Title
TaskTracker

## Problem Statement
Students and small teams need a lightweight, easy-to-use task tracker that provides simple analytics and helps estimate task duration for better planning.

## Scope
- Implement a web app supporting task lifecycle management and analytics.
- Provide documentation and required design artifacts as per guidelines.
- Include a simple ML predictor trained on synthetic data.

## Target Users
- Students, project teams, and learners.

## High-level Features
- User registration and login (demo)
- Task creation, update, deletion, completion tracking
- Analytics dashboard and report export
- Predictor endpoint for expected completion time
